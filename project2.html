Animal project (SWIFT)

This app is designed for young children (ages 2-5) to explore the sounds of different animals in a fun and interactive way.

Download Demo Video

Features

  • Colorful and visually appealing home screen design.
  • Interactive elements like buttons and animations.
  • Option to select different categories of animals (e.g., farm, wild, pets).
  • Clear navigation to other sections.

Animal Pages

  • Each page focuses on a single animal species.
  • Large, high-quality images of animals.
  • Clear and simple animal names (e.g., "Cow", "Lion", "Dog").
  • Audio playback button for each animal's sound.
  • Option to repeat the sound.

Technical Implementation

  • Swift Playground: Create a new playground project in Xcode.
  • Use Swift to implement image display and sound playback.
  • Design user interface elements using UIKit.
  • Include audio files for each animal's sound in appropriate formats.
