Project 3 - Portfolio Website

Portfolio Website Using HTML & CSS

This project involved creating a personal portfolio website using only HTML and CSS, without any external frameworks. It is hosted on GitHub Pages and showcases my skills in frontend development, particularly in creating a clean, responsive design.

The main goal was to improve my discipline in pure frontend development, focusing on usability, clean code, and well-structured content presentation. This portfolio serves as a showcase of my understanding of web development basics and how to organize a multi-page site that effectively presents information to visitors.

Technologies and Tools Used:

  • HTML for structuring the content
  • CSS for styling and layout
  • GitHub Pages for hosting the website
  • Flexbox for responsive design

Skills and Lessons Learned:

  • Improved discipline in writing clean, reusable code
  • Understanding of effective design principles for usability and readability
  • Knowledge of version control and project deployment via GitHub
  • Enhanced understanding of HTML semantics and CSS layout techniques
