Interactive Music School Platform

This project aims to create an interactive, user-friendly platform designed for a school of music. The application will be built using Java JavaFX, HTML, and CSS.

Download Structure Diagram
Download Introduction Video

Key Features

Song Searcher

  • Allows students to search for songs in the school's file registry by title, artist, genre, or instrument.
  • Displays search results with relevant information, including song details, artist, and instrument.

Tablature Storage

  • Provides a secure storage system, allowing students to upload and manage their tablatures.
  • Enables collaborative work by allowing students to share tablatures with others.

Interactive Learning Section

  • Offers in-depth visual guides and diagrams illustrating the workings of various instruments.

Fretboard Tablature Editor

  • Enables students to create and edit tablatures directly on a virtual fretboard.
  • Allows students to visualize and practice tablatures in real-time.

Interactivity

The platform is completely interactive, allowing students to:

  • Search for songs with ease.
  • Upload, manage, and share their tablatures.
  • Explore interactive diagrams and tutorials.
  • Create and edit tablatures using the virtual fretboard editor.
